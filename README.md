
# ⚛️ Modern Task App

A clean and intuitive **Task Management Application** built with **React.js**. This project demonstrates the power of reactive programming, focusing on a fluid user experience and robust component architecture.

---

## 🏗️ Architecture & Logic

- **Functional Components**: Built using modern React patterns with Hooks (`useState`, `useEffect`).
- **Reactive State**: Real-time UI updates triggered by state changes without page reloads.
- **Data Persistence**: Uses `localStorage` to ensure your tasks are saved even after closing the browser.
- **Modular UI**: A component-based structure that allows for easy scaling and maintenance.

---

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Language**: JavaScript (ES6+)
- **Styling**: CSS3 / Tailwind CSS
- **Build Tool**: Create React App / Vite

---

## 🚀 Key Features

- **Intuitive Task Entry**: Rapidly add and organize your daily goals.
- **Dynamic Filtering**: Easily toggle between active and completed tasks.
- **Interactive Progress**: Visual cues for task completion status.
- **Mobile Optimized**: Fully responsive design for productivity on the go.

---

## 💻 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone [https://github.com/shashidas95/todo-app.git](https://github.com/shashidas95/todo-app.git)
   cd todo-app
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Launch Development Server**:
   ```bash
   npm start
   ```
   *The app will be available at `http://localhost:3000`.*

---

## 📂 Project Structure

- `src/components/`: Reusable UI elements (Buttons, Input, List items).
- `src/hooks/`: Custom logic for state and persistence.
- `src/App.js`: Main application orchestrator.
